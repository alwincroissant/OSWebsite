PART 9: WINDOWS DEPLOYMENT SERVICES

Reference Playlist: https://www.youtube.com/playlist?list=PLN0M7olJoF6uJ98efXE4pIqhMX_M2GtrC

Use Windows Deployment Services to distribute Windows XP and Windows 7 images to requesting client computers over the network.

WDS Requirements
  • Install and configure Windows Deployment Services on Windows Server 2008.
  • Prepare boot and install images for Windows XP and Windows 7.
  • Allow PXE boot clients to request and receive OS deployment.
  • Deploy the correct OS image based on client request and lab requirement.
