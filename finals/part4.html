PART 4: FILE SHARING

Reference Video: https://youtu.be/C53gVyXxE4Y

Configure shared folders, permissions, and quotas so departments and offices can access only their assigned resources.

File Sharing Requirements
  • Each department and section must have one public folder with read/write access for assigned domain users.
  • Domain users cannot access folders of other departments.
  • Create a university public folder with read/write access for all domain users.
  • Create an installer folder for applications that only administrators can access.
  • All folders except the computer engineering folder should contain office documents only (PPT, DOC, DOCX, XLS, PPTX, PDF, etc.).
  • Information Technology folder should allow all files except executable files.
  • Set storage quota of 1 GB per folder.
  • Map shared folders on each client workstation.
