Windows Deployment Services (WDS)

Windows Deployment Services (WDS) is a server role in Microsoft Windows Server that enables network-based installation of Windows operating systems. It allows administrators to deploy Windows images to client computers over the network using PXE (Preboot Execution Environment) boot, eliminating the need for physical installation media. WDS streamlines OS deployment by centralizing image management, supporting unattended installations, and integrating with Microsoft Deployment Toolkit (MDT) for advanced automation. This makes it an essential tool for large-scale deployments, reducing deployment time and ensuring consistency across multiple systems.

Instructions:
- Install and configure Windows Deployment Services
- Add Windows images for network deployment
- Configure PXE boot settings
- Test deployment to client machines
