PART 8: GROUP POLICIES

Reference Playlist: https://www.youtube.com/playlist?list=PLN0M7olJoF6uckcDC6nSdP4-UXh8Xzm0T

Apply and enforce Group Policy Objects for user restrictions, security settings, password policy, account lockout, and interface controls.

Group Policy Requirements
  • Each department should have its own desktop wallpaper.
  • Enable: Prohibit access to Control Panel and PC settings.
  • Enable: Network security - do not store LAN Manager hash value on next password change.
  • Enable: Prevent access to the command prompt.
  • Enable: No auto-restart with logged on users for scheduled automatic updates installations.
  • Enable: All removable storage classes - deny all access.
  • Enable: Prohibit user install.
  • Disable: Accounts - Guest account status.
  • Minimum password length: at least 12 characters.
  • Maximum password age: 42 days.
  • Audit directory service access and directory service changes. GPO settings should only be accessed by IT admins.
  • Account lockout duration: 30 minutes.
  • Account lockout threshold: 3 invalid logon attempts.
  • Reset account lockout counter after: 30 minutes.
  • Remove links and access to Windows Update: user enabled, admin disabled.
  • Remove Network Connections from Start Menu: user enabled, admin disabled.
  • Remove Help Menu from Start Menu: user enabled, admin disabled.
  • Remove Run Menu from Start Menu: user enabled, admin disabled.
  • Remove My Pictures icon from Start Menu: user enabled, admin disabled.
  • Remove My Music icon from Start Menu: user enabled, admin disabled.
  • Remove My Network Places icon from Start Menu: user enabled, admin disabled.
  • Add Logoff on Start Menu: enabled.
  • Remove Task Manager from Ctrl+Alt+Del: user enabled, admin disabled.
  • Internet Explorer browser title text: set to domain name.
  • Prohibit access to LAN connection properties: user enabled, admin disabled.
  • Prevent access to command prompt: user enabled, admin disabled.
  • Prevent access to registry editing tools: user enabled, admin disabled.
  • Prohibit access to Control Panel: user enabled, admin disabled.
