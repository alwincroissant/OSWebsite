PART 3: DHCP SERVER

Reference Video: https://youtu.be/ngQXdd0AvTM

Deploy DHCP on Windows Server 2008 and configure scopes based on the required addressing plan.

DHCP Requirements
  • Use IP blocks: 172.31.0.0/16 and 10.0.0.0/8.
  • Set a valid router IP address in scope options.
  • DNS server IP should be the same as the DHCP server.
  • Also add Google DNS and OpenDNS public DNS server IP addresses.
  • Set lease duration to 8 hours.
  • Exclude the first 30 and last 30 valid IP addresses from distribution.
